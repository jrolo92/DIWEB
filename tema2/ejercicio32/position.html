static: Es el valor por defecto; el elemento se coloca según el flujo normal del documento, sin posicionamiento especial.
relative: El elemento se posiciona respecto a su lugar original, pudiendo moverse con top, left, right, bottom.
absolute: El elemento se posiciona respecto al primer contenedor con posición distinta de static, y se saca del flujo normal.
fixed: El elemento se posiciona respecto a la ventana del navegador y permanece fijo aunque se haga scroll.
sticky: El elemento se comporta como relative hasta que se alcanza un umbral de scroll, momento en que se queda fijo.
