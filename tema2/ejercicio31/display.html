CSS: Display
Línea oculta con display:none

La propiedad display en CSS define cómo un elemento se representa en la página y determina su comportamiento dentro del flujo del documento. block → El elemento ocupa todo el ancho y empieza en una nueva línea. inline → Se coloca en la misma línea y solo ocupa el espacio de su contenido. inline-block → Se mantiene en línea pero permite definir ancho y alto. none → El elemento no se muestra y no ocupa espacio.

La propiedad visibility controla si un elemento es visible o invisible, pero sigue ocupando espacio en el flujo del documento.

visibility: visible; → el elemento se muestra normalmente.

visibility: hidden; → el elemento no se ve, pero su espacio sigue reservado en la página. Es como si fuera transparente.
