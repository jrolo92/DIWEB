# Unidades de medida
1. ¿Cómo afecta cada unidad (vh, vw, %) al diseño cuando se cambia el tamaño de la ventana? \
Afecta de manera que se va adaptando el tamaño de cada elemento, haciendose mas pequeño o mas grande.
2. ¿Qué diferencias observas entre el comportamiento del .vh-box y el .vw-box? \
El vw-box es mas ancho, pero ambos tienen una anchura relativa.
3. ¿Cómo limita el tamaño del bloque .percent-box la propiedad max-width? \
Usa unidade absolutas de tipo píxeles.
